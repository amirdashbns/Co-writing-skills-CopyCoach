# Instagram Carousel Design Specifications — AI Copy Coach

## Canvas
- Size: 1080 x 1350 px (4:5 portrait)
- Background color: #FFF5F0 (warm off-white/blush)
- No background images or textures — always flat color

## Colors
- Primary text: #1A1A1A (near-black, softer than pure black)
- Accent text: #E63B2E (red/coral — used ONLY for emphasis words/phrases)
- Handle/arrow color: #3D2B1F (dark brown)
- Background: #FFF5F0

### Accent Color Rules
- Use red accent on 1-3 key phrases per slide MAX
- Never accent more than one line in a row
- Use it to highlight the "money words" — the phrase you want burned into their brain
- Never use red for full sentences — only fragments or key phrases

## Typography

### Headlines (hooks, bold statements, cover text)
- Font: Montserrat Black (or ExtraBold)
- Size: 64-80px for cover slides, 56-68px for inner slides
- Line height: 1.1 (tight — headlines should feel stacked and punchy)
- Letter spacing: -1px (slightly tight for impact)
- Transform: uppercase for cover slides and punch slides
- Alignment: left

### Body Text (explanations, stories, context)
- Font: DM Sans Regular (or Medium for slight emphasis)
- Size: 40-48px
- Line height: 1.4 (comfortable reading spacing)
- Letter spacing: 0
- Transform: none (sentence case)
- Alignment: left

### Handle (@aicopycoach)
- Font: DM Sans Medium
- Size: 28px
- Color: #3D2B1F
- Alignment: center
- Position: always at the bottom of every slide

## Spacing & Layout (THE KEY TO LOOKING PROFESSIONAL)

### Safe Zone / Padding
- Top padding: 100px
- Bottom padding: 160px (extra room for handle)
- Left padding: 80px
- Right padding: 80px
- This creates a "frame" — text never touches the edges

### Text Block Spacing
- Between headline and body text: 60px gap
- Between paragraphs: 40px gap
- Between body text and CTA/arrow: 60px gap
- Between last text element and @handle: auto (push handle to bottom)

### The "Breathing Room" Rules
1. NEVER fill more than 60% of the slide with text
2. If text takes up more than 60%, either:
   - Cut words (less is more)
   - Split into two slides
3. Every slide should have at least 25% white space
4. White space is not wasted space — it's what makes it look expensive

### Vertical Rhythm
- Content should sit in the upper 60-70% of the slide
- Bottom 30-40% is for: breathing room + arrow (if needed) + handle
- This creates a "gravity" that feels natural and clean

## Slide Type Templates

### TYPE 1: Cover Slide (first slide)
- Layout:
  - [100px top padding]
  - HEADLINE (Montserrat Black, 68-80px, uppercase)
  - [60px gap]
  - Illustration/icon (optional, centered, max 300x300px)
  - [60px gap]
  - SUBTITLE (Montserrat Bold Italic, 48-56px, mixed case)
  - [flexible space]
  - → arrow (bottom-right, 48px, color #3D2B1F)
  - [20px gap]
  - @aicopycoach (centered)
  - [40px bottom padding]

### TYPE 2: Story/Context Slide
- Layout:
  - [100px top padding]
  - Headline (Montserrat Bold, 52-60px, sentence case)
  - [60px gap]
  - Body text (DM Sans Regular, 40-44px)
  - Paragraph breaks: 40px between paragraphs
  - [flexible space]
  - @aicopycoach (centered)
  - [40px bottom padding]
- Rules:
  - Max 4-5 short paragraphs
  - Use red accent on 1-2 key phrases
  - No arrow on story slides (reader should stop and read)

### TYPE 3: Punch Slide (bold statement)
- Layout:
  - [120px top padding]
  - STATEMENT (Montserrat Black, 60-72px, uppercase)
  - Use red accent on the key phrase
  - [flexible space]
  - → arrow (bottom-right)
  - [20px gap]
  - @aicopycoach (centered)
  - [40px bottom padding]
- Rules:
  - MAX 3-4 lines of text
  - This slide should feel like a punch — short, bold, impactful
  - Lots of white space (40%+ of slide should be empty)

### TYPE 4: Body/Teaching Slide
- Layout:
  - [100px top padding]
  - Body text (DM Sans Medium, 42-48px)
  - Paragraph breaks: 40px
  - [flexible space]
  - @aicopycoach (centered)
  - [40px bottom padding]
- Rules:
  - Conversational tone
  - Max 5-6 short paragraphs
  - Can use bold (DM Sans Bold) for emphasis instead of red
  - No uppercase — this slide should feel like talking to a friend

### TYPE 5: CTA/Final Slide
- Layout:
  - [100px top padding]
  - Headline (Montserrat Bold, 56-64px): the ask or takeaway
  - [40px gap]
  - Body (DM Sans Regular, 40px): supporting context
  - [60px gap]
  - CTA line (Montserrat Bold, 44px, red accent)
  - [flexible space]
  - @aicopycoach (centered)
  - [40px bottom padding]

### TYPE 6: Numbered List/Tips Slide
- Layout:
  - [100px top padding]
  - Slide title (Montserrat Bold, 48px)
  - [40px gap]
  - Each item:
    - Number (Montserrat Black, 44px, red) + Text (DM Sans Medium, 40px)
    - 30px gap between items
  - [flexible space]
  - @aicopycoach (centered)
  - [40px bottom padding]
- Rules:
  - Max 4-5 items per slide
  - If more items, split across slides

## Logo Usage
- Logo file: Copy of AI Copy Coach Logo (2).png
- Use on: Cover slide only (optional) OR not at all (handle is sufficient)
- If used: max 120x120px, top-right or bottom-left corner
- Do NOT put logo on every slide — the handle does that job

## Swipe Arrow Rules
- Arrow: → (simple right arrow)
- Size: 48px line weight
- Color: #3D2B1F
- Position: bottom-right corner (80px from right edge, 180px from bottom)
- Use on: Cover slide, punch slides, transition slides
- Do NOT use on: Final/CTA slide, body/teaching slides where you want them to stop and read

## General Design Principles
1. Less text = more impact. When in doubt, cut words.
2. One idea per slide. If you're making two points, use two slides.
3. White space is your best friend. It's what separates "designed" from "made in Notes app."
4. Contrast creates hierarchy. Big + bold + red = look here first. Small + regular + black = read this second.
5. Consistency beats creativity. Same fonts, same spacing, same colors, every single carousel. That's how you build a recognizable brand.
6. Mobile-first always. If you can't read it on a phone without zooming, the text is too small or there's too much of it.
