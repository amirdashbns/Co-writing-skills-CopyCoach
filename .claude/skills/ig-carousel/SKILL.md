# Instagram Carousel Creator

Create scroll-stopping Instagram carousels that educate your audience, build authority, and drive saves, shares, and follows.

## Before Writing

1. **Read context profiles**:
   - `/context/voice-dna.json` - Match authentic voice
   - `/context/icp.json` - Write for target audience
   - `/context/business-profile.json` - Reference offerings when relevant

2. **Check knowledge base** for content to repurpose from `/knowledge/`

## Carousel Specifications

- **Slides**: 2–20 per carousel (Instagram limit)
- **Ideal length**: 7–10 slides for educational content
- **Aspect ratio**: 1080×1350 px (4:5 portrait) recommended; 1080×1080 (1:1 square) also common
- **Text per slide**: 30–80 words max (must be readable on mobile)
- **Font size guidance**: Headlines 24–32pt equivalent, body 16–20pt equivalent
- **Caption limit**: 2,200 characters

## Carousel Frameworks

### Framework 1: Problem → Agitate → Solve

```
Slide 1: [HOOK - State the problem boldly]
Slide 2: [AGITATE - Why this problem costs them]
Slide 3: [BRIDGE - There's a better way]
Slides 4–8: [SOLVE - Step-by-step solution, one step per slide]
Slide 9: [RECAP - Summarize the key steps]
Slide 10: [CTA - Save, share, follow, or link in bio]
```

### Framework 2: Listicle / Tips

```
Slide 1: [HOOK - "X things you need to know about..."]
Slides 2–8: [One tip per slide with brief explanation]
Slide 9: [BONUS TIP or recap]
Slide 10: [CTA]
```

### Framework 3: Myth vs. Reality

```
Slide 1: [HOOK - "Stop believing these myths about..."]
Slides 2–7: [MYTH on top → REALITY below, one per slide]
Slide 8: [KEY TAKEAWAY]
Slide 9: [CTA]
```

### Framework 4: Story / Case Study

```
Slide 1: [HOOK - Outcome or transformation teaser]
Slide 2: [THE STARTING POINT - Where things began]
Slides 3–7: [THE JOURNEY - Key moments, decisions, turning points]
Slide 8: [THE RESULT - Specific outcome with numbers]
Slide 9: [THE LESSON - What the audience can apply]
Slide 10: [CTA]
```

### Framework 5: Before vs. After

```
Slide 1: [HOOK - "What I changed to get..."]
Slide 2: [BEFORE state - Relatable struggle]
Slide 3: [THE SHIFT - What triggered change]
Slides 4–7: [KEY CHANGES - One per slide]
Slide 8: [AFTER state - The result]
Slide 9: [HOW TO START - Actionable first step for reader]
Slide 10: [CTA]
```

### Framework 6: How-To / Tutorial

```
Slide 1: [HOOK - "How to [achieve result] in [timeframe]"]
Slide 2: [WHAT YOU'LL NEED / CONTEXT]
Slides 3–8: [One step per slide with clear instruction]
Slide 9: [PRO TIP or common mistake to avoid]
Slide 10: [CTA]
```

### Framework 7: Contrarian / Hot Take

```
Slide 1: [HOOK - Bold contrarian statement]
Slide 2: [THE COMMON BELIEF you're challenging]
Slides 3–6: [YOUR ARGUMENT - Evidence and reasoning]
Slide 7: [WHAT TO DO INSTEAD]
Slide 8: [CTA - "Agree or disagree? Tell me below"]
```

## Writing Guidelines

### Cover Slide (Slide 1)

This determines whether people swipe. It must stop the scroll.

**Hook formulas:**
- "[Number] mistakes killing your [result]"
- "Stop doing [common thing] if you want [result]"
- "How I [achieved result] in [timeframe]"
- "The [topic] cheat sheet you didn't know you needed"
- "Read this before you [action they're about to take]"
- "Nobody talks about this [topic] truth"
- "[Result] isn't about [common belief]. It's about [real thing]."

**Design principles for slide 1:**
- One dominant headline (large, bold text)
- Minimal visual clutter
- High contrast for readability
- Optional: small subheadline or category label

### Body Slides (Slides 2–N)

- **One idea per slide** — never cram two concepts together
- **Headline + supporting text** structure on each slide
- Short sentences and sentence fragments work best
- Use numbered steps when showing a process
- Include specific examples, numbers, or screenshots where possible
- Maintain consistent visual style across all slides

### Final Slide (CTA)

Options:
- "Save this for later" (drives saves)
- "Share with someone who needs this" (drives shares)
- "Follow @handle for more [topic]" (drives follows)
- "Link in bio for [resource]" (drives traffic)
- "Which tip was your favorite? Comment below" (drives comments)
- Combine 2–3 CTAs (e.g., save + follow + comment)

### Caption Writing

The caption accompanies the carousel and appears below the post.

**Structure:**
```
[HOOK - First line must compel them to read more]

[CONTEXT - Why you created this carousel / personal angle]

[VALUE ADD - Extra tip not in the slides or expand on one point]

[CTA - Specific engagement ask]

[HASHTAGS - 5-15 relevant hashtags]
```

**Caption guidelines:**
- First line is critical (only ~125 characters show before "more")
- Use line breaks for readability
- Include a CTA that complements the slide CTA
- Add relevant hashtags (mix of broad and niche)

## Batch Generation

When asked for multiple carousel ideas:

1. Provide 5–10 cover slide hook options
2. Note which framework each would use
3. Let user select favorites
4. Develop full slide decks from chosen hooks

## Repurposing for Carousels

When converting long-form content:

1. Identify the core thesis or transformation
2. Extract 5–8 key points or steps
3. Write a scroll-stopping cover slide hook
4. One point per slide with clear headline + brief body
5. Add a recap or summary slide
6. Write a CTA slide
7. Draft a caption that adds context beyond the slides

## Carousel Quality Checklist

Before delivering:

- [ ] Cover slide stops the scroll
- [ ] Voice matches voice DNA
- [ ] Each slide has one clear idea
- [ ] Text is readable at mobile size (not too dense)
- [ ] Consistent visual structure across slides
- [ ] Logical flow from slide to slide
- [ ] ICP would find this valuable enough to save
- [ ] CTA is clear and specific
- [ ] Caption adds value beyond the slides
- [ ] Hashtags are relevant and well-mixed

## What to Avoid

- Too much text on any single slide (walls of text)
- Inconsistent formatting between slides
- Weak or vague cover slides
- No clear CTA
- Generic advice that could come from anyone
- Slides that don't make sense without the caption
- Purely promotional carousels with no value
- Using all 20 slides when 8 would do (respect attention spans)

## Example Carousel

### Topic: "How to Write Faster"

```
📌 SLIDE 1 (Cover)
Headline: I write 3x faster than I did a year ago.
Subheadline: Here are the 5 changes I made.

📌 SLIDE 2
Headline: Change #1: I stopped editing while writing.
Body: First draft = brain dump. Editing comes later. Mixing the two slows everything down.

📌 SLIDE 3
Headline: Change #2: I use voice memos.
Body: I talk through ideas on walks, transcribe them, then clean up. My best writing starts as speech.

📌 SLIDE 4
Headline: Change #3: I write at the same time every day.
Body: 6:30–8:00 AM. No exceptions. Consistency beats motivation every time.

📌 SLIDE 5
Headline: Change #4: I keep an idea bank.
Body: I never sit down to a blank page. I capture ideas all week and pick the best one when it's time to write.

📌 SLIDE 6
Headline: Change #5: I set a timer.
Body: 25-minute sprints. No phone. No tabs. Constraints create focus.

📌 SLIDE 7
Headline: Quick recap:
Body:
1. Separate writing from editing
2. Use voice memos
3. Write at a consistent time
4. Keep an idea bank
5. Use timed sprints

📌 SLIDE 8 (CTA)
Headline: Save this for your next writing session.
Body: Follow @handle for more writing tips that actually work.
```
